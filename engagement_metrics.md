# Engagement Metrics For Gamification

Goal:

Measure whether gamification improves worker participation.

## Metrics To Track

### 1. Daily Activity Rate
- Number of active workers per day
- Sessions completed

### 2. Task Completion Rate
- Household registrations completed
- Immunization tracking completed
- Screening tasks completed

### 3. XP Progress
- XP earned daily
- XP earned weekly

### 4. Streak Retention
- Consecutive active days
- Streak loss rate

### 5. Badge Unlock Frequency
- Most earned badges
- Rare badges

### 6. Engagement Trend
- Before gamification
- After gamification

## Firebase Analytics Events

Examples:

- badge_unlocked
- xp_earned
- streak_completed
- daily_goal_completed

## Success Indicators

- Increased task completion
- Reduced inactive users
- Higher daily usage
- Better data consistency
