# FLW Gamification Problem Understanding

The FLW Mobile App is used by ASHA workers for:

- Household registration
- Pregnancy tracking
- Child health monitoring
- Immunization tracking
- NCD screening

Current Problem:

The app has no engagement system.

Workers perform repetitive tasks daily.

Possible issues:

- Low motivation
- Reduced consistency
- Data fatigue
- Less engagement

Goal:

Build a gamification module to improve:

- Motivation
- Completion rate
- Data quality
- Habit formation
